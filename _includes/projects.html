Point of Sale system designed for Raspberry Pi - I worked on creating a Point of Sales (POS) system at Realbox focus on restaurants that want to move online. The POS was focused on online orders management and was designed to run a raspberry Pi for reasonable load.

Basic Audio based Ontology Analysis - I completed this project as part of my college's final major project. We used Google's recently released audioset, sonic-annotator for feature extraction, feature ranking and selection and finally Amazon machine learning to accelerate the process.
An OS based on Rust - I completed this project as part of the minor project submission for my college's degree. We used rust to write a barebones 64 bit OS. The goal was to investigate the potential exploitation of GPUs (We had to use Tesla series card, since Geforce series cards don't support privileged instructions). There were some very interesting results and many key learnings.
